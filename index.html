Software

A SAVE possui um software atualizado de nome VS(Vacancy Special), com facilidade para nossos usuários,com interface dinâmica, que permite realizar cadastros, editar cadastros para se ter um controle da parte Administrativa.

Restrição de Vagas

A necessidade das restrições das vagads, é necessária para que os direitos sejam preservados, através de uma cancela eletrônica a SAVE impedirá que pessoas que não tem o direito, estacionem.

Facilidade de Uso

A facilidade que o SAVE proporciona é muito alta, pois além de ser um facilitador da mobilidade de pessoas com deficiência, ele é também é um gerenciador de Estacionamento, com interface dinâmica e um Suporte Offline completo.

Sobre

Conheça um pouco mais sobre o SAVE

  • 1ºSemestre de 2015

    Como surgiu

    O SAVE surgiu com a idéia de defender os direitos dos deficientes físicos, e idosos que necessitam de vagas especiais. A equipe continua firme e forte, disposta a implantar este sistema no meio real. A idéia foi de acordo com todos os integrantees, que se empenharam para realizá-lo.

  • 2ºSemestre de 2015

    Planejando e Organizando

    Através do SAVE, pessoas que têm direito a vagas especiais, terão suas vagas reservadas, pela cancela e para poder adentra-las terão que estar com o cartão em mãos, aproximar o mesmo para leitura do código, consecutivamente a liberação da vaga será efetuada.

  • Desenvolvimento

    Resultados Excitantes

    Os resultados no fim do projeto foram animadores pois recebemos diversos elogios por ter criado o sistema que finalmente irá ajudar a restringir todas as vagas especiais.

  • 2ºSemestre de 2015

    Conclusão

    A Conclusão do projeto foi um sucesso, pois o auxílio a milhares de pessoas portadoras de alguma deficiência física e idosa, agora é possível.

Leitor RFID

Tecnologia de última geração.

O leitor é o componente de comunicação entre o sistema RFID e os sistemas externos de processamento de informações. A complexidade dos leitores depende do tipo de etiqueta e das funções a serem aplicadas. Os mais sofisticados apresentam funções de verificação de paridade de erro e correção de dados. Uma vez que os sinais do receptor sejam corretamente recebidos e decodificados, são usados algoritmos para decidir se o sinal é uma repetição de transmissão de uma etiqueta.
